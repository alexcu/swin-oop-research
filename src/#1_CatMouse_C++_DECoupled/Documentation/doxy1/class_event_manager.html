Detailed Description

Defines EventManager class which processes each event to each kind of EventProcessor.

Author
Alex Cummaudo
Date
7 Oct 2013
Note
This is a static member class; so that clients do not need to make an instance of an EventManager (since there's only ever going to be one processor). Therefore we invoke EventManager by calling directly on the class (i.e. EventManager::notify_subscribers(event))
